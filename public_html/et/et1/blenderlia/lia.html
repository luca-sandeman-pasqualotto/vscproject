For the final project, I decided I want to create an animation in blender. This animation is going to be based off Animal Crossing, using a redesigned version of the model I used for the first blender project.

My main goal for this project will be properly implementing armatures into my models and making fluid animations that resemble the game my character comes from.

For the animation in question, I was choosing between having a simple demonstration of the character showing off different expressions, like you can do in the game. My other idea was the character trying to catch a fish, and either showing it off, or falling while trying to catch it.
